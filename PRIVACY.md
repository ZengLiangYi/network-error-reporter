# Privacy

Network Error Reporter 在本地 Chrome DevTools 面板内运行，不会主动将数据发送到任何远程服务。

## 数据范围

- 读取当前 DevTools 会话中的网络请求信息
- 读取当前页面 URL
- 生成本地 Markdown 报告和本地图片导出

## 数据处理方式

- 所有数据处理都发生在本地浏览器扩展环境内
- 不包含账号系统、云同步或远程日志上传
- 不收集分析数据、遥测数据或行为埋点

## 用户可控操作

- 报告仅在用户手动点击复制或导出时输出
- 如果报告中包含敏感字段，扩展会在复制前给出提醒
- 导出图片失败时，扩展仅会在本地下载图片文件，不会上传到任何服务
